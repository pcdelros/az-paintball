Form security notes

Recommended

  • Validate on the server, not only the browser
  • Add spam protection and rate limiting
  • Collect only what is needed to respond

Avoid

  • Email passwords in JavaScript
  • Private API keys in frontend files
  • Hidden admin links
  • Unclear fake-submit behavior
